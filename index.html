Welcome to the Cryptology Companion!

The Cryptology Companion - CryptoCompanion for short - is a collection various cryptography techniques and ciphers. It also includes code files and examples so that you can try out these cryptographic methods for yourself. This site was made by Andrew Zhou for the 2017 PA High School Computer Fair's "Web Page Design" Competition. It is an expansion of an App Inventor project that he did for AP Computer Science Principles. It was created from scratch using HTML, CSS, and Bootstrap. The CryptoCompanion logo was made using Inkscape. Other media are sourced from Wikimedia Commons and Google Fonts.

Use the navigation bar to explore the website.
